# Curso JavaScript e TypeScript - Full Stack 🚀

Este repositório contém meus estudos, exercícios e projetos desenvolvidos durante o curso **"JavaScript e TypeScript - front-end e back-end (Full Stack)"** do instrutor Luiz Otávio Miranda.

O foco é dominar desde a lógica de programação pura até o desenvolvimento de aplicações robustas e escaláveis.

## 🛠️ Tecnologias e Ferramentas
* **Linguagens:** JavaScript (ES6+) e TypeScript
* **Ambiente:** Node.js, Express, noSQL (MongoDB), SQL (PostgreSQL)
* **Front-end:** React.js, Hooks, Redux
* **Infraestrutura:** Git/GitHub, WSL2 (Ubuntu), Docker
* **Padrões:** Design Patterns, SOLID

## 📂 Organização do Repositório
* `/logica-programacao`: Exercícios de algoritmos e fundamentos.
* `/javascript-basico-avancado`: Funções, Objetos, Prototypes e Assincronismo.
* `/typescript`: Implementação de tipagem e POO.
* `/backend`: APIs desenvolvidas com Node.js e Express.
* `/frontend`: Interfaces criadas com React e estilização.
